  • This option will upload your Prospective Excel File, read the data and insert into the database.
  • About Excel Sheet formatting, this option has the following requirements:
  • The Worksheet name must be named 'Sheet1'
  • The OrderItem Id should be in row 2, and column 1, value in col 3
  • Source tag should be in row 2 col 3, value in column 4
  • The file must be an Excel Workbook, with XLSX extension. Column Titles should be in row 4
  • Data should begin from row 5th row.

  • The Worksheet name must be named 'Sheet1'
